# PanoptesPDU

## Trivia

Argus is a character described in the greek mythology but also the name of the PDUs deployed in our hardware lab.
He had the byname Panoptes which is derived from the hundred eyes in his head and all over his body.
As this repo serves to host an open source alternative firmware for said PDUs, it seemed as a fitting name.

## The Hardware

The Argus PDU is a rather simple device, as it mainly consits of a single logic IC that switches relais by toggling GPIOs.
The IC is a PIC18F92J60 Family chip, namely the PIC18F67J60 variant.
This chip has a 10 MBits ethernet interface to on which it serves a web server that hosts a web UI and API to interface with the PDU.
