# CroPDUster

## Trivia

A cropduster is a type of old agricultural airplane used to supply crops with fertilizers.
Not only is it one of the only words in the dictionary having the letters PDU in it to form a nice pun, but it also easy to draw a logo for it.

## The Hardware

The Argus PDU is a rather simple device, as it mainly consits of a single logic IC that switches 8 relais by toggling GPIOs.
The IC is a PIC18F92J60 Family chip, namely the PIC18F67J60 variant.
This chip has a 10 MBits ethernet interface to on which it serves a web server that hosts a web UI and API to interface with the PDU.
It also has a 7 segment display to show information, a buzzer and humidity and temperature sensor.
